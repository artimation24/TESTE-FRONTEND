# TESTE-FRONTEND
Este é meu projeto de tela de espera de jogo multiplayer.

Primeiro, gostaria de confessar o uso de IA generativa para auxílio na produção do código, porém isso foi apenas pelo fato de que não daria tempo de eu aprender tudo o que é necessário para a realização do projeto, mas também garanto que aproveitei o código gerado para estudar e aprender como fazer por conta própria no futuro, tendo feito algumas modificações no código original conforme ia testando e aprendendo.

Compreendo que isto pode custar minha vaga no estágio, e aceitarei sem problemas caso seja recusado por causa deste uso de IA.

Com isso esclarecido, vamos aos detalhes do projeto.

## Como rodar o projeto
O meu projeto utiliza apenas HTML, CSS e JavaScript, então os arquivos podem ser baixados e rodados no navegador com o arquivo index.html, pode ser rodado pelo próprio GitHub Pages, e não é necessário nenhum dev server ou algo do tipo.

## Tecnologias utilizadas
O código foi feito e testado através da plataforma online CodeSandbox, e gerado com ajuda do ChatGPT. Eu solicitei ao ChatGPT um código base e fui alterando ele com formatos de botões, cores, bordas de botões e cards, texturas de botões desabilitados, mudanças essas feitas e testadas através do CodeSandbox.

## Decisões técnicas
A estrutura visual foi significativamente simplificada em comparação com o exemplo fornecido nas especificações por escolha de deixar o visual o mais limpo possível, além de que algumas mudanças e detalhamentos trouxeram problemas que não tiveram tanto tempo para serem resolvidos. Eu também achei melhor adicionar comentários nos arquivos para facilitar minha organização e a identificação de quais partes do código fazem o que.

## Melhorias que faria com mais tempo
A primeira e mais notável é fazer o código inteiro sem nenhum IA generativa. Eu também acrescentaria alguns detalhes e mudaria algumas fontes para uma personalização melhor e para alguns upgrades de UI. Com mais tempo, eu também iria me dispor a aprender React para melhorar o projeto e expandir meus próprios conhecimentos.
